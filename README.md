CNCL project Businesses For You
Problem: Many local businesses struggle to gain visibility. This platform helps increase exposure by allowing users to easily discover, review, and save businesses in one place.
Businesses For You is a web application that helps users discover local businesses. Users can filter by category, sort by rating, leave reviews, and save their favorite businesses.

The platforms goal is to boost engagement between local small businneses and youth, giving an easy way for users to explore and see deals.
